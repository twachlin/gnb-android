### Goliath National Bank (GNB) - App

This is an Android application and was made using:

Architecture:
- MVVM architecture
- Dependency Injection design pattern

Data:
- Retrofit
- Moshi

Ui:
- Jetpack Compose
- LiveData

Testing:
- Junit
- MockK

This repository was made following Trunk-Based Development (TBD) strategy
